## Description

This project is an educational web application built with pure HTML, CSS, and JavaScript (no frameworks). It covers essential front-end concepts like selectors, box model, flexbox, transitions, and more. The site supports English and Uzbek languages, dark/light modes, and includes an interactive exam to test knowledge.

Ideal for beginners learning front-end development or preparing for exams. The content is sourced from provided documents and organized into navigable sections.
